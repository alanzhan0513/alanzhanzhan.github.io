最近剛好在更換域名，所以順手做一下筆記，也讓大家可以輕鬆上手，設定自己的 Github page 域名。

第一步：購買域名

先去購買自己想要的域名名稱：我是去 Google Domain 購買的，如果你有喜歡或者偏好的域名供應商，你也可以去你想要的供應商購買，本篇就以 Google Domain 為例。

第二步：DNS 設定中新增 CNAME 紀錄

購買完域名後，需要設定 DNS 相關的設定，可以參照下圖，跟著設定就好。

第三步：設定 Github Page

在你的網站的根目錄新增一個 CNAME 的檔案，內容就放你的域名名稱即可

1

alanzhan.dev

並且到 Github 上設定你的域名名稱

第四步：等待

等待一段時間等待 DNS 紀錄生效！

搞定收工下課！
